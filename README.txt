PIA CNN
Este trabajo trata acerca de una red comvolucional que se trata de predecir si una imagen de cáncer de piel es de tipo benigno o maligno, el cual fue entrenado con un 
conjunto de entrenamiento de imagenes y otro conjunto de prueba. Siendo porbado con un conjunto de imagenes de validación

FORMA DE EJEUCIÓN
Para ejecutar este proyecto, se debe seguir los siguientes pasos:
1. Tner instalado, Python, Anaconda, Tensorflow, Keras, Matplotlib y PIL (para ver y ejecutar el código se puede usar Visual Code o Notebook Jupiter).
2. Descargar el conjunto de imagenes del siguiente link 
3. Ejecutar el programa llamado Training.py o Training.ipynb.
4. Ya ejecutado el programa anterior, ejecutar el programa llamado Classifier.py o Classifier.ipynb

Los archivos deben estar organizados en la misma carpeta de la siguiente manera para poder ejecutarse sin errores:
- data (carpeta con imagenes)
- Training.py
- Classifier.py
- Training.ipynb
- Classifier.ipynb
